Skincare Website

This was the first informational website I created for my preliminary
project in my third year's first semester at Southville. I utilized HTML,
CSS, and JavaScript for this project. The site features five different
pages, as shown in the screenshots. You begin at the login page, and
if you log in as a Regular User, you will see the standard color scheme
of the random Pokémon assigned by our professor. If the user logs in
as a Shiny User, the website will display the shiny version's color
scheme of the Pokémon. Additionally, this website incorporates jQuery
animations that I implemented for buttons and page transitions.

8-bit Binary Game

This was a project I developed during my second year, featuring an
8-bit binary game where the player inputs their name to get started.
Players are presented with a number that they must convert into binary
before the timer expires. As shown in the screenshot, my design follows
an ice cream theme. If the binary value is 0, the ice cream cone appears
empty, but a value of 1 indicates that the cone has a scoop of ice cream.
Once the time is up, the player's score is displayed on the leaderboard
along with their total playtime.
